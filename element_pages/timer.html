Timer
Explanation
This element provides a class to take times in some many units
Units 
        
  enum class Precision
  {
    minutes = 0,
    seconds,
    milliseconds,
    microseconds,
    nanoseconds,
    picoseconds
  };
Functions
Like a timer, this element can start, stop, continue & see time. 
        
  class TimeCont
  {
    void startTime();
    void continueTime();
    void stopTime();

    size_t getElapsedTime(Precision precision);
    static inline void Delay(s32 milliseconds);
  }
Example
To use the timer is easy. You need to start the timer, if you call stopTime the time stops & you can call continueTime to keep it 
        
  void main() 
  {
	  TimeCont timer;

    timer.startTime();
    TimeCont::Delay(16); // In milliseconds -> 16 ms
    timer.stopTime();
    timer.printElapsedTime(TimeCont::Precision::milliseconds);
    // The console may print Time: 16ms

    TimeCont::Delay(20);
    timer.printElapsedTime(TimeCont::Precision::milliseconds);
    // The console may print Time: 16ms

    timer.continueTime();
    TimeCont::Delay(16);
    timer.printElapsedTime(TimeCont::Precision::milliseconds);
    // The console may print Time: 32ms
  }
